# 情绪场景库 · Emotional Scene Library

**不教你怎么写。帮你看清从哪里写。**

这是一个 Claude Skill。它做两件事：

1. 帮你建立一份你自己的**张力档案**——在五种核心感觉（爱、失控、羞耻、被抛弃、欲望）上，看清你身上四股力的拉扯方向。
2. 在你讨论世界观、人物、关系、细纲的全过程中，带着这份档案**持续帮你校验**——不是替你写，是在你要走向套路的时候，把你自己的路摆出来。

---

## 为什么做这个

写小说最难的不是技巧。是你说不清自己到底想写什么。

你知道套路里那个不是你的，但你也说不出你要的是哪个。你给AI描述角色，它写出来干净利落，但你读了觉得不对——不是不好，是太扁了。

因为一个人想写的东西从来不是一个单一的方向。它至少有四股力在同时拉扯：

- **你"是"什么样的** — 你的真实运行方式
- **你"愿意是"什么样的** — 你认领的那个版本
- **你"希望是"什么样的** — 你渴望但还做不到的方向
- **你"拒绝是"什么样的** — 你绝不愿意成为的那种人

这四股力之间的矛盾，就是你写作的全部空间。这个Skill帮你看清它们，然后在写作全过程中用它们。

## 安装

### Claude（推荐）

下载 `emotional-scene-library.skill` 文件，然后：

1. 打开 [claude.ai](https://claude.ai)
2. 点击左下角头像 → 自定义 → Skills
3. 上传 `.skill` 文件
4. 确保 Skill 已开启

### 其他AI工具（GPT、豆包、Kimi等）

把 `SKILL.md` 的全部内容复制粘贴到你的系统提示词/自定义指令里。如果需要某个具体模块，可以单独复制 `references/` 文件夹里的对应文件。

---

## 使用

### 第一次：建档

打开Claude，直接说：

> "我想建立我的张力档案"

Skill会在五个维度上各问你4-5个问题，大约20-30分钟。走完后生成一份结构化的张力档案。**把这份档案复制保存下来。**

### 之后每次：共创

开新对话时，把你的张力档案贴在开头，然后直接聊你的写作问题：

> [贴上张力档案]
>
> "我在写一个现代言情，男主的人设是这样的……你觉得怎么样？"

AI会带着你的档案底色回应。当你做出一个"套路"选择的时候，它会告诉你——你的档案里有一条不一样的路。

### 卡在某场具体的戏

不需要建档也能用。直接说：

> "我卡在一场背叛揭露的戏上，女主发现男主骗了她，我不知道她该怎么反应"

Skill会进入对应的卡点场景提问，帮你找到属于你的写法。

## 覆盖的场景

### 现代言情卡点（11个）

| 阶段 | 场景 |
|------|------|
| 还没开始 | 暗恋 · 试探 · 表白 |
| 关系裂缝 | 第一次争吵 · 冷战 · 背叛 |
| 分离 | 主动离开 · 被离开 |
| 回来 | 重逢 · 和解 · 破镜重圆 |

爱 · 失控 · 羞耻 · 被抛弃 · 欲望

## 核心理念

**你的小说世界之所以"薄"，不是缺设定，是缺矛盾。**

不是剧情的矛盾。是你自己身上的矛盾。你跟爱的关系方式、你跟失控的关系方式、你跟羞耻的关系方式——这些东西里面的四股力拉扯，就是你故事的气候来源。

这个Skill不生产内容。它帮你看清你自己身上已有的东西，然后让AI在整个创作过程中记住它、使用它、在你忘记的时候提醒你。
