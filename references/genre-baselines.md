# 类型文基准线与最佳偏离节点

## 什么是基准线

基准线是读者进入一个类型时默认期待的情节走法和角色反应。它不是"烂俗"——它是类型之所以成为类型的骨架。

偏离不是反套路。好的偏离是：在读者期待最强烈的节点上，给出一个他从没见过但比他期待的更对的反应。

## 偏离的原则

1. 不是所有节点都值得偏离。过度偏离会让读者困惑。选择偏离回报最高的节点。
2. 偏离必须基于用户的张力场，不能凭空编造。
3. 最好的偏离让读者觉得"原来还可以这样写"，而不是"这个作者在标新立异"。
4. 偏离通常发生在角色的行动选择上，而不是情节走向上。情节可以走套路，但角色在套路节点上的反应方式是你的。

## 现代言情

### 甜文基准线

| 节点 | 基准反应 | 偏离回报 |
|------|---------|---------|
| 心动瞬间 | 脸红/心跳加速/移开视线 | 中 |
| 暧昧期互动 | 制造偶遇/借口靠近 | 低 |
| 表白 | 直球告白/被逼问后承认 | 高 |
| 第一次亲密接触 | 紧张/僵住/对方主导 | 中 |
| 确认关系后日常 | 撒娇/互怼/投喂 | 低 |
| 误会 | 不解释/第三者介入/巧合 | 高 |
| 和好 | 一方先低头/真相大白 | 高 |

高偏离回报节点：表白、误会、和好。这些是读者期待最具体的地方，偏离会产生最大的"陌生但愉快"效果。

### 暗恋文基准线

| 节点 | 基准反应 | 偏离回报 |
|------|---------|---------|
| 暗恋的表现 | 偷看/记住对方习惯/帮忙 | 中 |
| 被发现了 | 否认/尴尬/逃避 | 高 |
| 对方有别的暧昧对象 | 吃醋但压住/默默退出 | 中 |
| 终于说出口 | 被逼到极限才说/酒后真言 | 高 |
| 对方回应 | 其实也喜欢/需要时间 | 低 |

### 破镜重圆基准线

| 节点 | 基准反应 | 偏离回报 |
|------|---------|---------|
| 分手原因 | 误会/身份差距/第三者 | 低 |
| 分手时的反应 | 决绝转身/强忍泪水 | 高 |
| 分手后状态 | 表面放下实际没有/借酒消愁 | 中 |
| 重逢 | 偶遇/装作不认识/装作不在乎 | 高 |
| 重逢后互动 | 冷淡中带裂缝/忍不住关心 | 中 |
| 真相揭露 | 发现当年另有隐情 | 低 |
| 再次靠近 | 一方先软化/旧情复燃 | 中 |
| 复合决定 | 放下过去重新开始 | 高 |

### 追妻火葬场基准线

| 节点 | 基准反应 | 偏离回报 |
|------|---------|---------|
| 前期虐待/忽视 | 男主冷漠/不珍惜 | 低 |
| 女主离开 | 净身出户/消失/重生 | 中 |
| 男主意识到 | 发现离不开/后悔 | 高 |
| 追妻阶段 | 跪求/付出/改变 | 高 |
| 女主态度 | 冷淡/不原谅/犹豫 | 高 |
| 男主证明 | 关键时刻牺牲/持续付出 | 中 |
| 和解 | 女主终于心软/原谅 | 高 |

追妻文的高偏离节点集中在"男主意识到""追妻方式""女主态度"三个地方。这三个节点的默认反应最固定，偏离空间最大。

## 使用方式

当用户走完某个场景的提问并生成了场景卡片后，如果他/她需要"偏离建议"：

1. 先找到对应的类型基准线
2. 定位他/她正在写的节点
3. 看该节点的偏离回报等级
4. 用他/她的张力场（特别是"你是"和"你拒绝是"之间的矛盾）去替换基准反应
5. 生成2-3个偏离方案，标注每个方案更靠近用户四个方向中的哪一个

好的偏离通常来自用户"你是"和"你拒绝是"之间的张力——因为那个张力本身就是最不寻常的地方，是最难被别人复制的。
